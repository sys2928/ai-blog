## Perplexity AI란?

Perplexity는 AI와 검색 엔진을 결합한 도구입니다. 질문하면 웹을 실시간으로 검색하고, 그 결과를 바탕으로 정확하고 출처가 명확한 답변을 줍니다.

ChatGPT와 가장 큰 차이점은 **최신 정보에 접근**할 수 있다는 점입니다.

## 언제 Perplexity를 써야 할까?

**Perplexity가 유리한 경우**
- 오늘 뉴스나 최신 정보 검색
- 특정 사실 확인이 필요할 때
- 출처가 필요한 조사/리서치
- 제품 가격, 현재 이벤트 확인

**ChatGPT/Claude가 유리한 경우**
- 창의적 글쓰기
- 복잡한 분석과 추론
- 코딩
- 긴 문서 처리

## 핵심 기능

**Focus 기능**
검색 범위를 지정할 수 있습니다.
- Web: 전체 웹 검색
- Academic: 학술 논문 검색
- YouTube: 유튜브 영상 내용 검색
- Reddit: 커뮤니티 의견 검색

**출처 명시**
모든 답변에 출처 링크가 달려 있어 사실 확인이 쉽습니다.

**후속 질문**
대화 형식으로 계속 물어볼 수 있습니다.

## 실제 활용 예시

**블로그 리서치**
```
오늘 기준으로 가장 주목받는 AI 도구 3가지와 이유를 알려줘
```

**경쟁 분석**
```
[회사명]의 최근 6개월 주요 소식과 제품 업데이트 정리해줘
```

**시장 조사**
```
한국 AI 스타트업 시장 현황 2026년 기준으로 알려줘
```
