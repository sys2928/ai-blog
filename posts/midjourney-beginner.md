## Midjourney란?

Midjourney는 텍스트를 입력하면 고품질 이미지를 생성해주는 AI 도구입니다. 디자인 경험이 없어도 원하는 이미지를 만들 수 있어요.

## 시작하기 전에 필요한 것

- Discord 계정 (무료)
- Midjourney 구독 ($10/월부터)

## 설치 및 시작 방법

**1단계: Discord 설치**
discord.com 에서 Discord를 다운로드하고 계정을 만드세요.

**2단계: Midjourney 서버 접속**
midjourney.com 접속 → "Join the Beta" 클릭 → Discord에서 Midjourney 서버 입장

**3단계: 구독 설정**
`/subscribe` 명령어 입력 → 플랜 선택 → 결제

**4단계: 첫 이미지 생성**
newbies 채널에서 `/imagine` 입력 후 원하는 이미지 설명 작성

## 기본 프롬프트 작성법

```
/imagine prompt: [주제], [스타일], [분위기], [색감], [화질]
```

예시:
```
/imagine prompt: 한옥 카페 내부, 따뜻한 조명, 오후의 햇살, 아늑한 분위기, 8K
```

## 자주 쓰는 파라미터

| 파라미터 | 설명 | 예시 |
|---|---|---|
| --ar | 이미지 비율 | --ar 16:9 |
| --v 6 | 최신 버전 | --v 6 |
| --style raw | 사실적 스타일 | --style raw |
| --q 2 | 최고 품질 | --q 2 |

## 초보자가 자주 하는 실수

**너무 짧은 프롬프트**: "고양이"보다 "흰색 페르시안 고양이, 창가에 앉아있는, 부드러운 오후 햇살, 수채화 스타일"처럼 구체적으로.

**스타일 혼재**: 사실적 스타일과 만화 스타일을 동시에 요청하면 어중간한 결과가 나옵니다.

## 유용한 스타일 키워드

- `photorealistic` — 사진처럼 사실적
- `watercolor` — 수채화
- `oil painting` — 유화
- `minimalist` — 미니멀
- `cinematic lighting` — 영화같은 조명
- `studio lighting` — 스튜디오 조명
