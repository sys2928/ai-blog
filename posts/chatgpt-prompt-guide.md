## 왜 프롬프트가 중요한가?

ChatGPT는 질문을 어떻게 하느냐에 따라 결과물이 완전히 달라집니다. 같은 AI인데 누군가는 엄청난 결과를 얻고, 누군가는 실망하는 이유가 바로 프롬프트 차이입니다.

## 업무 생산성 템플릿

**이메일 작성**
```
다음 상황에 맞는 정중한 비즈니스 이메일을 작성해줘.
상황: [상황 설명]
받는 사람: [직함/관계]
핵심 내용: [전달할 내용]
톤: 공식적이지만 친근하게
```

**회의록 요약**
```
아래 회의 내용을 요약해줘.
형식: 1) 주요 결정사항 2) 액션 아이템 (담당자, 기한 포함) 3) 다음 회의 안건
[회의 내용 붙여넣기]
```

**보고서 초안**
```
[주제]에 대한 보고서 초안을 작성해줘.
독자: [대상]
목적: [목적]
분량: A4 2장 분량
포함할 내용: [핵심 포인트들]
```

## 글쓰기 템플릿

**블로그 글**
```
[주제]에 대한 블로그 글을 써줘.
독자: [타겟 독자]
톤: [전문적/친근한/유머러스]
포함할 키워드: [SEO 키워드]
분량: 800~1000자
소제목 3개 이상 포함
```

**SNS 콘텐츠**
```
아래 내용을 인스타그램/링크드인용 게시물로 바꿔줘.
원본: [내용]
해시태그 5개 포함
이모지 적절히 사용
```

## 분석 및 리서치 템플릿

**경쟁사 분석**
```
[회사명]의 마케팅 전략을 분석해줘.
분석 항목: 타겟 고객, 주요 메시지, 채널 전략, 강점/약점
표 형식으로 정리해줘
```

**아이디어 브레인스토밍**
```
[주제]에 대한 아이디어를 20개 생성해줘.
제약 없이 창의적으로
각 아이디어는 한 줄로 요약
```

## 학습 템플릿

**개념 설명**
```
[개념]을 설명해줘.
조건: 초등학생도 이해할 수 있게, 실생활 예시 3개 포함, 5분 안에 읽을 수 있는 분량
```

**퀴즈 생성**
```
[주제]에 대한 퀴즈 10개를 만들어줘.
형식: 객관식 4지선다
난이도: 중급
정답과 해설 포함
```

## 핵심 팁

프롬프트를 잘 쓰는 핵심은 **역할 + 상황 + 형식 + 제약**을 명확히 하는 것입니다. 막연한 질문보다 구체적인 조건을 줄수록 원하는 결과에 훨씬 가까워집니다.
