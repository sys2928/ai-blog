## 유튜브 대본, AI로 이렇게 씁니다

영상 기획은 할 수 있는데 대본 쓰는 게 막막하다는 분들이 많아요. ChatGPT를 잘 활용하면 아이디어에서 완성 대본까지 10분이면 충분합니다.

## 1단계 — 주제와 구조 잡기

먼저 영상의 뼈대를 잡습니다.
```
다음 유튜브 영상의 구성을 짜줘.
주제: [영상 주제]
타겟: [시청자 연령/관심사]
영상 길이: [목표 분량]
채널 톤: [전문적/친근한/유머러스]

포함할 내용:
- 후킹 인트로 (30초)
- 본론 3~4개 섹션
- 아웃트로 + CTA
```

## 2단계 — 후킹 인트로 작성

첫 15초가 구독자를 잡느냐 마느냐를 결정합니다.
```
이 유튜브 영상의 후킹 인트로를 3가지 버전으로 써줘.
주제: [주제]
1) 충격적인 통계/사실로 시작
2) 공감되는 문제 상황으로 시작
3) 질문으로 시작
30초 분량 (약 75단어)
```

## 3단계 — 본문 대본 작성

구조가 잡히면 섹션별로 작성합니다.
```
위 구성의 [섹션명] 부분 대본을 써줘.
조건:
- 구어체로 자연스럽게
- 중간에 시청자에게 말 거는 부분 포함
- 예시나 비유 포함
- 분량: 약 [X]분 (1분 = 약 150단어)
```

## 4단계 — 섬네일 문구와 제목 뽑기
```
이 영상의 유튜브 제목 10개를 써줘.
- 숫자 포함 버전
- 질문형 버전
- 혜택 강조 버전
- 호기심 자극 버전

섬네일에 들어갈 짧은 문구(10자 이내)도 5개 제안해줘
```

## 5단계 — 설명란과 해시태그
```
이 영상의 유튜브 설명란을 써줘.
- 영상 요약 (3줄)
- 타임스탬프
- 채널 구독 유도 문구
해시태그 15개 포함
```

## 실제 활용 팁

**나만의 스타일 학습시키기**: 기존에 잘 된 영상 대본을 ChatGPT에 보여주고 "이 스타일로 써줘"라고 하면 채널 고유의 톤을 유지할 수 있어요.

**반복 구조 만들기**: 매주 비슷한 형식의 영상을 만든다면 템플릿 프롬프트를 저장해두고 주제만 바꿔서 사용하세요.

**검토는 필수**: AI가 쓴 대본은 반드시 소리 내어 읽어보세요. 자연스럽게 말할 수 있는지 확인하고 수정하는 게 중요해요.
