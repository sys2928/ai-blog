## GPTs가 뭔가요?

GPTs는 ChatGPT를 특정 목적에 맞게 커스텀한 버전입니다. OpenAI가 2023년 11월에 공개한 기능으로, 코딩 없이 누구나 만들 수 있어요.

쉽게 말하면 이런 겁니다.

- 일반 ChatGPT: 만능이지만 매번 상황을 설명해야 함
- GPTs: 특정 역할과 지시사항이 미리 세팅된 전문가 버전

예를 들어 "블로그 글 작성 도우미" GPT를 만들면, 매번 "너는 블로그 작가야, 톤은 친근하게..." 같은 프롬프트를 반복할 필요 없이 주제만 던지면 바로 원하는 스타일로 글을 써줍니다.

## GPTs는 어디서 쓸 수 있나요?

ChatGPT 왼쪽 사이드바에서 **"GPT 탐색"**을 클릭하면 GPT Store에 접속할 수 있어요.

**무료 사용자:** 다른 사람이 만든 GPTs를 사용할 수 있지만, 일일 사용 횟수에 제한이 있습니다.

**Plus 사용자 (월 $20):** GPTs 무제한 사용 + 직접 GPT 만들기가 가능합니다.

## 바로 써먹을 수 있는 GPTs 추천 8개

GPT Store에는 수만 개의 GPTs가 있지만, 실제로 쓸만한 건 일부입니다. 직접 써보고 괜찮았던 것만 골랐어요.

### 글쓰기/콘텐츠

**1. Write For Me**
용도: 블로그, 에세이, 보고서 등 긴 글 작성. 주제와 톤만 알려주면 구조화된 글을 바로 써줍니다. 한국어도 잘 됩니다.

**2. Humanizer**
용도: AI가 쓴 티가 나는 글을 자연스럽게 다듬어줍니다. AI 탐지 도구에 걸리지 않도록 문체를 바꿔주는 용도로 유용해요.

### 이미지/디자인

**3. DALL·E (공식)**
용도: 텍스트로 이미지 생성. ChatGPT에 내장된 것과 같지만, GPT 버전은 프롬프트 가이드가 더 상세합니다.

**4. Logo Creator**
용도: 브랜드 로고 디자인. 스타일, 색상, 분위기를 말하면 여러 시안을 만들어줍니다. 초기 스타트업이나 사이드 프로젝트에 유용해요.

### 업무/생산성

**5. Consensus**
용도: 학술 논문 검색 및 요약. 질문을 하면 관련 논문을 찾아서 근거 기반으로 답변해줍니다. 리서치할 때 시간을 많이 줄여줘요.

**6. Diagrams: Show Me**
용도: 플로우차트, 마인드맵, 시퀀스 다이어그램 등을 텍스트 설명만으로 생성해줍니다. 기획서나 발표 자료에 넣을 다이어그램 만들 때 좋아요.

### 개발/기술

**7. Grimoire**
용도: 코딩 도우미. 웹사이트, 앱, 스크립트를 대화로 만들 수 있어요. 코딩 초보도 간단한 웹페이지 정도는 만들 수 있습니다.

**8. SQL Expert**
용도: SQL 쿼리 작성 및 최적화. 테이블 구조를 알려주고 원하는 결과를 말하면 쿼리를 짜줍니다.

## 나만의 GPT 만드는 법 (5단계)

코딩 지식이 전혀 필요 없습니다. 대화로 만들 수 있어요.

### 1단계: GPT 만들기 시작

ChatGPT 왼쪽 사이드바 → **"GPT 탐색"** → 오른쪽 상단 **"+ 만들기"** 클릭.

### 2단계: 이름과 설명 정하기

GPT에 이름을 붙이고, 어떤 용도인지 한 줄로 설명합니다.

예시:
- 이름: "블로그 글 도우미"
- 설명: "SEO에 최적화된 한국어 블로그 글을 작성해주는 AI"

### 3단계: 지시사항(Instructions) 작성

이 부분이 GPT의 핵심입니다. 이 GPT가 어떤 역할을 하고, 어떻게 답변해야 하는지를 자세히 적어요.

**잘 쓴 지시사항 예시:**

```
너는 한국어 블로그 전문 작가야.

규칙:
1. 모든 글은 "~입니다" 체가 아닌 "~해요" 체로 작성
2. 도입부에 독자의 문제점이나 궁금증을 먼저 제시
3. 소제목은 최소 5개 이상
4. 각 섹션에 구체적인 예시 포함
5. 마지막에 핵심 요약을 3줄로 정리
6. SEO를 위해 키워드를 자연스럽게 배치
```

### 4단계: 대화 시작 문구 설정

사용자가 GPT를 열었을 때 보이는 예시 질문들을 설정할 수 있어요. 4개까지 가능합니다.

예시:
- "AI 도구 비교 글을 써줘"
- "이 초안을 SEO에 맞게 다듬어줘"
- "이 주제로 블로그 개요를 잡아줘"
- "이 글의 제목을 10개 만들어줘"

### 5단계: 지식 파일 업로드 (선택)

PDF, 텍스트 파일 등을 업로드하면 GPT가 그 내용을 참고해서 답변합니다. 예를 들어 회사 브랜드 가이드라인을 올려두면 그에 맞춰서 글을 써줘요.

설정이 끝나면 **"만들기"**를 누르면 완성입니다.

## GPT를 잘 만드는 3가지 팁

### 1. 지시사항은 구체적일수록 좋다

"글을 잘 써줘"보다 "~해요 체로, 소제목 5개 이상, 각 섹션 200자 이상, 예시 포함"처럼 세부 조건을 넣을수록 일관된 결과가 나와요.

### 2. 하지 말아야 할 것도 적어라

"이모지 사용하지 마", "영어 단어를 괄호로 병기하지 마", "인사말로 시작하지 마" 같은 금지 사항을 넣으면 불필요한 수정이 줄어듭니다.

### 3. 테스트하고 계속 다듬어라

한 번에 완벽한 GPT를 만들기는 어렵습니다. 실제로 써보면서 지시사항을 조금씩 수정해나가는 게 핵심이에요. "Configure" 탭에서 언제든 수정할 수 있어요.

## GPTs vs 프롬프트 저장, 뭐가 다를까?

"그냥 좋은 프롬프트를 메모장에 저장해두면 되지 않나?"라고 생각할 수 있어요. 차이가 있습니다.

**프롬프트 저장:** 매번 새 대화에서 복붙해야 하고, 이전 대화 맥락이 이어지지 않아요.

**GPTs:** 지시사항이 항상 적용되어 있어서 주제만 바로 던지면 되고, 업로드한 지식 파일도 항상 참조할 수 있어요. 반복 업무에서 시간 차이가 큽니다.

## 핵심 요약

- GPTs는 ChatGPT를 특정 용도에 맞게 커스텀한 버전으로, 코딩 없이 만들 수 있다
- GPT Store에서 바로 쓸 수 있는 전문 GPTs를 활용하면 업무 시간을 크게 줄일 수 있다
- 나만의 GPT를 만들 때는 지시사항을 구체적으로 쓰고, 실제로 써보면서 계속 다듬는 게 핵심이다
