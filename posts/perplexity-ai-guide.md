## Perplexity AI란?

Perplexity는 AI와 검색 엔진을 결합한 도구입니다. 질문하면 웹을 실시간으로 검색하고, 그 결과를 바탕으로 정확하고 출처가 명확한 답변을 줍니다.

ChatGPT와 가장 큰 차이점은 **최신 정보에 접근**할 수 있다는 점입니다. ChatGPT는 학습 데이터 기준으로 답변하지만, Perplexity는 지금 이 순간의 웹 정보를 검색해서 답변합니다. "오늘 애플 주가가 얼마야?" 같은 질문에 Perplexity는 바로 답해줘요.

## 언제 Perplexity를 써야 할까?

AI 도구마다 강점이 다릅니다. Perplexity는 "최신 정보 + 출처가 필요한 검색"에서 진가를 발휘해요.

**Perplexity가 유리한 경우:**
- 오늘 뉴스나 최신 정보 검색
- 특정 사실 확인이 필요할 때 (숫자, 날짜, 가격 등)
- 출처가 필요한 조사/리서치
- 여러 소스의 정보를 한 번에 종합할 때

**ChatGPT/Claude가 유리한 경우:**
- 창의적 글쓰기, 코딩, 긴 문서 처리

각 AI 도구의 차이가 궁금하다면 [Claude vs ChatGPT 비교](/posts/claude-vs-chatgpt)를 참고하세요.

## 핵심 기능

**Focus 기능 — 검색 범위 지정**

Perplexity의 가장 유용한 기능입니다. 검색 범위를 지정해서 원하는 소스에서만 정보를 가져올 수 있어요.

- **Web**: 전체 웹 검색 (기본)
- **Academic**: 학술 논문만 검색 — 리서치할 때 유용
- **YouTube**: 유튜브 영상 내용 검색 — 영상 속 정보를 텍스트로
- **Reddit**: 커뮤니티 의견 검색 — 실사용 후기 찾을 때

예를 들어 "ChatGPT 실제 사용 후기"를 Reddit Focus로 검색하면 실제 사용자들의 의견만 모아서 정리해줍니다.

**출처 명시**

모든 답변에 출처 링크가 달려 있어 사실 확인이 쉽습니다. 블로그 글이나 보고서를 쓸 때 출처를 밝혀야 하는 경우 매우 편리해요.

**후속 질문**

대화 형식으로 계속 물어볼 수 있습니다. "좀 더 자세히", "한국 시장 기준으로 다시 정리해줘" 같은 후속 질문이 자연스럽게 이어져요.

## 실제 활용 예시

**블로그 리서치:**
```
오늘 기준으로 가장 주목받는 AI 도구 3가지와 이유를 알려줘.
각 도구의 무료 플랜 기능도 포함해줘.
```

**경쟁 분석:**
```
[회사명]의 최근 6개월 주요 소식과 제품 업데이트를 표 형식으로 정리해줘.
```

**학습/리서치:**
```
[주제]에 대한 최신 학술 연구를 Academic Focus로 검색해줘.
2025년 이후 논문 중심으로.
```

저도 블로그 글을 쓸 때 주제 리서치 단계에서 Perplexity를 가장 먼저 씁니다. 전체 워크플로우가 궁금하면 [AI로 블로그 글 쓰는 워크플로우](/posts/ai-writing-workflow)를 확인하세요.
