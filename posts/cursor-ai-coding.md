## Cursor AI란?

Cursor는 AI가 내장된 코드 에디터입니다. VS Code 기반이라 기존 VS Code 사용자라면 바로 적응할 수 있습니다.

GitHub Copilot과 다른 점은 코드 전체 컨텍스트를 AI가 이해하고, 자연어로 대화하며 코딩할 수 있다는 점입니다.

## 설치 방법

1. cursor.com 접속
2. 운영체제에 맞는 버전 다운로드
3. VS Code 설정 가져오기 (기존 확장 프로그램, 테마 그대로 이전)

## 핵심 기능 3가지

**1. Tab 자동완성**
코드를 타이핑하면 다음 줄을 예측해서 자동 완성합니다. Tab 키만 누르면 적용.

**2. Cmd+K (코드 생성/수정)**
선택한 코드를 자연어로 수정할 수 있습니다.
```
"이 함수에 에러 핸들링 추가해줘"
"TypeScript로 변환해줘"
"성능 최적화해줘"
```

**3. Cmd+L (AI 채팅)**
코드 전체를 컨텍스트로 AI와 대화합니다.
```
"이 파일에서 버그를 찾아줘"
"이 API를 어떻게 개선할 수 있어?"
"테스트 코드 작성해줘"
```

## 실제 업무에서 이렇게 씁니다

**반복 코드 작성**: "이 패턴으로 나머지 5개 함수도 만들어줘"

**레거시 코드 리팩토링**: 오래된 코드 선택 → "현대적인 패턴으로 리팩토링해줘"

**문서화**: 함수 선택 → "JSDoc 주석 추가해줘"

**디버깅**: 에러 메시지 붙여넣기 → "이 에러 원인과 해결 방법 알려줘"
