## 자소서, AI로 써도 되나?

결론부터 말하면, **AI가 대신 써주는 게 아니라 AI를 도구로 활용하는 것**입니다. AI한테 "자소서 써줘"라고 하고 그대로 복붙하면 면접관이 바로 알아챕니다. 모든 지원자의 자소서가 비슷해지거든요.

하지만 이렇게 쓰면 다릅니다. 내 경험을 정리하고 → AI가 구조를 잡아주고 → 내가 다시 다듬는 방식. 처음부터 빈 페이지에서 시작하는 것보다 훨씬 빠르고, 결과물도 좋아요.

## AI 자소서 활용의 올바른 흐름

**내 경험/강점 정리 (직접)** → **AI로 구조화 + 초안** → **내 언어로 수정 (직접)** → **AI로 검토/다듬기**

핵심은 처음(재료)과 마지막(수정)은 내가 하고, 중간 과정에서 AI를 활용하는 겁니다.

## 템플릿 1 — 경험 정리하기

자소서를 쓰기 전에 먼저 내 경험을 정리해야 합니다. 이걸 AI에게 맡기면 안 돼요. 하지만 정리하는 "틀"은 AI가 도와줄 수 있습니다.

```
나는 [직무]에 지원하려고 해.
아래 질문에 대한 답을 정리하면 자소서 재료가 될 거야.
질문만 만들어줘 (답은 내가 쓸게).

포함할 항목:
- 이 직무에 관심을 갖게 된 계기
- 관련 경험 (프로젝트, 인턴, 동아리 등)
- 문제를 해결한 구체적 사례
- 나만의 강점과 그 근거
- 이 회사를 선택한 이유
```

AI가 만든 질문에 직접 답을 적으세요. 이게 자소서의 재료가 됩니다.

## 템플릿 2 — 초안 구조 잡기

재료가 준비되면 AI에게 구조화를 요청합니다.

```
아래는 내가 자소서에 쓸 경험과 강점이야.
이걸 바탕으로 [회사명] [직무]에 맞는 자기소개서 초안을 써줘.

[위에서 정리한 답변 붙여넣기]

조건:
- 분량: 각 항목 300~500자
- 구성: 지원 동기 / 관련 경험 / 강점 / 입사 후 포부
- 톤: 진솔하고 구체적으로
- 추상적인 표현("열정적인", "성실한") 대신 구체적 사례와 수치 사용
- "~한 사람입니다" 같은 뻔한 결론 피하기
```

**중요:** AI가 만든 초안을 그대로 쓰면 안 됩니다. 이건 뼈대일 뿐이에요. 여기에 내 목소리를 입히는 게 다음 단계입니다.

## 템플릿 3 — 회사 맞춤화

같은 자소서를 여러 회사에 돌리면 금방 티가 납니다. 회사별로 맞춤화할 때 AI가 유용해요.

```
아래 자기소개서를 [회사명]에 맞게 수정해줘.

[현재 자소서 붙여넣기]

이 회사의 특징:
- 업종: [업종]
- 인재상: [회사 인재상 — 채용 페이지에서 확인]
- 최근 주요 사업: [최근 뉴스나 사업 방향]

회사의 인재상과 사업 방향에 자연스럽게 연결해줘.
억지스럽지 않게, 내 경험과의 접점 위주로.
```

회사 정보를 찾을 때는 [Perplexity AI](/posts/perplexity-ai-guide)로 최신 뉴스와 채용 정보를 검색하면 빨라요.

## 템플릿 4 — 자소서 검토/피드백

초안을 다듬은 후, AI에게 검토를 요청합니다.

```
아래 자기소개서를 채용 담당자 관점에서 검토해줘.

[자소서 붙여넣기]

검토 항목:
1) 구체성: 추상적인 표현이 남아있는지
2) 설득력: "그래서 왜 이 사람을 뽑아야 하는지" 느껴지는지
3) 차별화: 다른 지원자와 구분되는 포인트가 있는지
4) 문장력: 어색하거나 반복되는 표현
5) 분량: 각 항목별 적절한지

수정 제안을 구체적으로 해줘.
```

## 템플릿 5 — 면접 대비 질문 생성

자소서를 기반으로 면접 질문까지 미리 준비할 수 있어요.

```
아래 자기소개서를 바탕으로 면접에서 나올 만한 질문 10개를 만들어줘.

[자소서 붙여넣기]

질문 유형:
- 경험 검증 질문 3개 ("~한 경험에 대해 자세히 말해보세요")
- 압박 질문 3개 ("~하면 어떻게 하시겠어요?")
- 가치관 질문 2개 ("왜 이 직무를 선택했나요?")
- 역량 질문 2개 ("본인의 약점은 무엇인가요?")

각 질문에 대한 답변 포인트도 간단히 제안해줘.
```

## 절대 하면 안 되는 것

**AI 결과를 그대로 복붙하지 마세요.** 채용 담당자들은 AI가 쓴 글을 점점 잘 구분합니다. "도전적인 환경에서 성장하고 싶습니다", "팀워크를 통해 시너지를 창출하겠습니다" 같은 AI 특유의 뻔한 표현이 남아있으면 바로 티가 나요.

**반드시 내 경험을 구체적으로 넣으세요.** "3개월간 인스타그램 팔로워를 200명에서 1,500명으로 늘렸습니다"처럼 숫자와 구체적 사실이 있어야 합니다. 이 부분은 AI가 만들어줄 수 없고, 오직 본인만 쓸 수 있어요.

**여러 회사에 같은 자소서를 내지 마세요.** 위 템플릿 3을 활용해서 회사별로 꼭 맞춤화하세요.
