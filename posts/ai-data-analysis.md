## 엑셀 함수 몰라도 데이터 분석할 수 있다

"VLOOKUP이 뭐야?", "피벗 테이블 어떻게 만들어?" — 이런 고민 때문에 데이터 분석을 포기한 적 있나요?

AI를 쓰면 엑셀 함수를 몰라도 됩니다. 데이터를 ChatGPT에 복붙하고 "이 데이터에서 뭐가 보여?"라고 물어보면 AI가 분석해줍니다. 함수를 외우는 대신, 질문하는 법만 알면 돼요.

## 방법 1 — 데이터를 직접 복붙하기

엑셀이나 구글 시트에서 데이터를 선택 → 복사 → ChatGPT에 붙여넣으면 됩니다.

```
아래 매출 데이터를 분석해줘.

이름, 1월, 2월, 3월
김철수, 120, 150, 180
이영희, 200, 180, 210
박민수, 90, 110, 95
정수진, 160, 170, 190

분석 항목:
1) 전체 트렌드 요약
2) 가장 성장률이 높은 사람
3) 주의가 필요한 사람
4) 다음 달 예상 매출
```

ChatGPT가 숫자를 읽고 트렌드, 성장률, 이상값까지 분석해줍니다. 엑셀 함수 하나 안 쓰고요.

## 방법 2 — 파일 업로드하기 (ChatGPT Plus)

ChatGPT Plus 사용자라면 엑셀/CSV 파일을 직접 업로드할 수 있어요.

```
이 엑셀 파일을 분석해줘.
[파일 업로드]

다음 항목을 알려줘:
1) 전체 요약 (3줄)
2) 상위 5개 항목과 하위 5개 항목
3) 월별 트렌드 차트
4) 이상값이나 특이점
```

파일을 올리면 Code Interpreter가 파이썬으로 자동 분석하고 차트까지 그려줍니다. 무료vs유료 차이가 궁금하면 [ChatGPT 무료 vs 유료 비교](/posts/chatgpt-free-vs-plus)를 참고하세요.

## 방법 3 — 차트/그래프 요청하기

숫자만 보면 감이 안 올 때, 시각화를 요청하세요.

```
위 데이터로 다음 차트를 만들어줘:
1) 월별 매출 추이 꺾은선 그래프 (사람별로 색상 구분)
2) 1분기 총 매출 막대 그래프 (사람별)
3) 매출 비중 파이 차트
```

ChatGPT Plus의 Code Interpreter가 실제 차트를 생성해서 보여줍니다.

## 방법 4 — 설문조사 결과 분석

```
아래는 고객 만족도 설문 결과야. 분석해줘.

[설문 데이터 붙여넣기]

분석 항목:
1) 전체 만족도 평균
2) 항목별 만족도 순위
3) 가장 불만이 높은 항목과 개선 제안
4) 긍정/부정 키워드 분석
```

## 방법 5 — 경쟁사 비교 분석

```
아래 3개 회사의 데이터를 비교 분석해줘.

[데이터 붙여넣기]

비교 항목:
1) 각 회사 강점/약점
2) 성장률 비교
3) 우리 회사가 집중해야 할 영역
표 형식으로 정리해줘.
```

## 실전 팁 — 더 좋은 분석을 받으려면

**1. 데이터와 함께 맥락을 알려주세요**

"매출 데이터야"보다 "B2B SaaS 회사의 월별 매출 데이터야. 신규 고객 유치 캠페인을 3월에 시작했어"라고 하면 AI가 맥락을 고려한 분석을 해줘요.

**2. 원하는 결과 형식을 지정하세요**

"표로 정리해줘", "차트로 만들어줘", "한 문단으로 요약해줘" 등 형식을 말해주면 바로 쓸 수 있는 결과가 나옵니다.

**3. 후속 질문을 하세요**

첫 분석을 받고 "왜 3월에 떨어졌을까?", "이 트렌드가 계속되면 6월 예상치는?", "경쟁사 대비 어떤 편이야?" 같은 후속 질문을 하면 더 깊은 인사이트를 얻을 수 있어요.

**4. 민감한 데이터 주의**

고객 개인정보, 회사 기밀 데이터는 ChatGPT에 넣지 마세요. 민감하지 않은 데이터만 사용하거나, 개인정보를 제거한 후 분석하세요.

## AI 분석의 한계

AI 분석은 빠르고 편하지만 100% 정확하지는 않습니다. 특히 숫자 계산에서 가끔 실수할 수 있어요. 중요한 의사결정에 쓸 데이터는 AI 분석 결과를 반드시 직접 검증하세요.
