## ChatGPT, 뭘 물어봐야 할지 모르겠다면

ChatGPT를 처음 열면 빈 입력창만 덩그러니 있어요. "뭘 물어봐야 하지?" "이런 것도 물어봐도 되나?" 하는 생각이 들기 마련입니다.

사실 ChatGPT는 뭘 물어보느냐보다 **어떻게 물어보느냐**가 더 중요해요. 같은 질문이라도 말하는 방식에 따라 유용한 답이 나올 수도 있고, 뻔한 답이 나올 수도 있습니다.

이 글은 ChatGPT를 처음 쓰는 분을 위한 기본 대화법입니다. 어려운 기술이 아니라, 일상 대화에서 약간의 습관만 바꾸면 되는 수준이에요.

## 1. 구체적으로 물어보세요

가장 흔한 실수는 너무 넓게 물어보는 거예요.

**이렇게 하지 마세요:**
```
영어 공부 방법 알려줘
```
→ "단어를 외우세요, 문법을 공부하세요" 같은 누구나 아는 답이 나옵니다.

**이렇게 하세요:**
```
직장인인데 매일 30분씩 영어 공부하고 싶어. 
회화 위주로, 3개월 안에 해외 출장에서 기본 대화가 가능한 수준이 목표야. 
주간 학습 계획을 짜줘.
```
→ 내 상황에 맞는 구체적인 학습 계획이 나옵니다.

핵심은 **누구인지, 무엇을 원하는지, 어떤 조건이 있는지**를 알려주는 거예요. 친구한테 부탁할 때도 "뭐 좀 알려줘"보다 "이런 상황인데 이거 도와줘"가 더 좋은 답을 얻잖아요. ChatGPT도 마찬가지입니다.

## 2. 한 번에 하나만 물어보세요

여러 질문을 한꺼번에 던지면 ChatGPT가 앞쪽 질문에 집중하고 뒤쪽은 대충 넘어갈 수 있어요.

**이렇게 하지 마세요:**
```
이직 준비 어떻게 해? 이력서는 어떻게 써? 면접은 어떻게 준비해? 연봉 협상은?
```

**이렇게 하세요:**
```
이직 준비를 시작하려고 해. 먼저 이력서부터 쓰고 싶은데, 
3년차 마케터가 이직할 때 이력서에 꼭 넣어야 할 항목이 뭐야?
```
→ 이력서 답변을 받고 → 그 다음 면접 질문 → 그 다음 연봉 협상. 순서대로 하나씩 물어보면 각 답변의 퀄리티가 올라갑니다.

## 3. "~처럼 해줘"가 강력합니다

원하는 스타일이나 수준을 레퍼런스로 알려주면 결과가 확 달라져요.

```
초등학생도 이해할 수 있게 설명해줘
```

```
TED 강연처럼 흥미롭게 써줘
```

```
친구한테 카톡 보내듯이 캐주얼하게 써줘
```

```
보고서 형식으로 정리해줘
```

ChatGPT는 톤과 수준을 맞추는 능력이 뛰어나요. "전문가처럼", "5살 아이에게 설명하듯이", "뉴스 기사처럼" 등 원하는 스타일을 말해주세요.

## 4. 마음에 안 들면 다시 시키세요

첫 번째 답변이 마음에 안 드는 건 정상이에요. ChatGPT는 대화를 이어갈수록 내가 원하는 걸 더 잘 파악합니다.

자주 쓰는 피드백 문장들:

```
좋은데 너무 길어. 핵심만 3줄로 줄여줘.
```

```
톤이 너무 딱딱해. 친구한테 말하듯이 바꿔줘.
```

```
2번 항목이 좀 약한데, 구체적인 예시를 추가해줘.
```

```
다른 각도에서 다시 써줘.
```

한 번에 완벽한 답을 기대하지 마세요. 2~3번 피드백을 주고받으면 원하는 수준의 결과가 나옵니다.

## 5. 결과 형식을 미리 정해주세요

같은 내용이라도 형식에 따라 쓸모가 달라집니다.

```
표로 정리해줘
```
→ 비교할 때 유용

```
번호 리스트로 정리해줘
```
→ 순서가 있는 작업에 유용

```
한 문단으로 요약해줘
```
→ 보고서나 이메일에 바로 복붙 가능

```
장점/단점으로 나눠서 정리해줘
```
→ 의사결정할 때 유용

형식을 지정하면 ChatGPT 답변을 후처리하는 시간이 크게 줄어요.

## 6. 이어서 대화하세요

ChatGPT는 같은 대화 안에서 이전 내용을 기억합니다. 새 채팅을 열 필요 없이 이어서 물어보세요.

예를 들어:
1. "여행 계획 짜줘" → 일정 받음
2. "2일차를 좀 더 여유롭게 바꿔줘" → 수정된 일정
3. "이 일정 기준으로 예산도 계산해줘" → 예산 추가
4. "영어로 번역해줘" → 영문 버전

하나의 대화에서 계속 발전시키는 게 매번 새로 시작하는 것보다 훨씬 좋은 결과를 줍니다.

## 7. "잘 모르겠어"도 괜찮아요

ChatGPT에게 방향을 잡아달라고 해도 됩니다.

```
이직을 고민 중인데 뭐부터 해야 할지 모르겠어. 
내가 생각해봐야 할 질문들을 먼저 정리해줘.
```

```
블로그를 시작하고 싶은데 주제를 못 정했어. 
내가 관심 있는 분야가 [분야]인데, 어떤 주제가 좋을지 10개 제안해줘.
```

뭘 물어봐야 할지 모를 때, "질문을 만들어달라"고 하는 것도 좋은 방법이에요.

## 초보자가 바로 써먹는 대화 예시 5개

**일상:**
```
이번 주말에 서울에서 데이트하려고 해. 
실내 활동 위주로, 예산 5만원 이내, 코스 추천해줘.
```

**업무:**
```
팀장님한테 보낼 이메일을 써줘. 
프로젝트 일정이 1주일 늦어질 것 같은데, 정중하게 알리는 내용으로.
```

**학습:**
```
파이썬을 처음 배우려고 해. 
완전 초보인데, 첫 1주일 동안 뭘 하면 되는지 알려줘.
```

**쇼핑:**
```
노트북을 사려는데 예산 100만원이야. 
대학생이 쓰기 좋은 가성비 노트북 3개만 추천해줘. 비교표로.
```

**건강:**
```
하루 종일 앉아서 일하는 직장인인데, 
점심시간 15분 동안 할 수 있는 스트레칭 루틴 짜줘.
```
