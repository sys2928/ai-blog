## 왜 요약에 ChatGPT를 써야 할까?

10페이지짜리 보고서를 30초 만에 핵심만 파악하고 싶을 때가 있어요. 회의 전에 자료를 빠르게 훑어야 하거나, 긴 기사의 요점만 알고 싶을 때. ChatGPT에 통째로 복붙하고 "요약해줘"라고 하면 되지만, 어떻게 요청하느냐에 따라 결과가 크게 달라집니다.

"요약해줘"만 쓰면 AI가 알아서 판단한 요약이 나오고, 프롬프트를 구체적으로 쓰면 내가 원하는 형태의 요약이 나와요. 아래 상황별 프롬프트를 복붙해서 써보세요.

## 상황 1 — 빠르게 핵심만 파악하기

```
아래 글을 3줄로 요약해줘.
각 줄은 한 문장으로, 가장 중요한 내용 순서로.

[글 붙여넣기]
```

가장 기본적인 요약이에요. 회의 전에 자료를 빠르게 훑을 때, 기사의 요점만 알고 싶을 때 씁니다.

## 상황 2 — 보고서용 구조화 요약

```
아래 내용을 다음 형식으로 요약해줘.

1) 핵심 결론 (1줄)
2) 주요 근거 3가지 (번호 리스트)
3) 실행 필요 사항 / 액션 아이템 (있으면)
4) 한계점 또는 주의사항 (있으면)

[글 붙여넣기]
```

상사에게 보고하거나, 팀에 공유할 때 유용해요. 결론부터 나오니까 읽는 사람이 바로 핵심을 파악할 수 있습니다.

## 상황 3 — 여러 문서 비교 요약

```
아래 2개 글을 읽고 비교 요약해줘.

비교 항목:
- 각 글의 핵심 주장
- 공통점
- 차이점
- 어떤 상황에서 어떤 글의 관점이 더 유용한지

글 1: [붙여넣기]
글 2: [붙여넣기]
```

두 보고서, 두 기사, 두 논문의 차이를 빠르게 파악할 때 쓰면 좋아요.

## 상황 4 — 회의록 요약

```
아래 회의 내용을 요약해줘.

형식:
- 주요 결정사항 (번호)
- 액션 아이템 (담당자, 기한 포함 — 표로)
- 미결 이슈
- 다음 회의 안건

[회의 내용 붙여넣기]
```

회의 중에 대충 메모한 것도 깔끔한 회의록으로 바꿔줍니다. [ChatGPT로 업무 생산성 높이는 방법](/posts/chatgpt-productivity)에서 더 많은 업무 활용법을 확인하세요.

## 상황 5 — 초등학생도 이해할 수 있게

```
아래 내용을 초등학생도 이해할 수 있게 쉬운 말로 요약해줘.
전문 용어는 쉬운 단어로 바꾸고, 비유를 사용해줘.
분량: 5줄 이내.

[글 붙여넣기]
```

어려운 논문이나 기술 문서의 핵심을 파악할 때, 또는 비전공자에게 설명해야 할 때 유용해요.

## 상황 6 — 영어 글 한국어로 요약

```
아래 영문을 한국어로 요약해줘.
번역이 아니라 한국어로 된 요약이 필요해.
핵심 포인트 5개를 번호 리스트로.

[영어 글 붙여넣기]
```

영어 기사나 논문을 읽을 시간이 없을 때, 번역 없이 바로 한국어 요약을 받을 수 있어요. AI 번역에 관심 있다면 [AI 번역 도구 비교](/posts/ai-translation-tools)도 참고하세요.

## 요약 퀄리티를 높이는 팁

**1. 원하는 분량을 지정하세요.** "3줄로", "200자 이내로", "A4 반 페이지로" 등 분량을 정하면 너무 길거나 짧은 요약을 피할 수 있어요.

**2. 형식을 지정하세요.** "번호 리스트로", "표로", "한 문단으로" 등 원하는 형태를 말해주면 바로 쓸 수 있는 결과가 나옵니다.

**3. 독자를 알려주세요.** "임원진이 읽을 거야", "비전공자 대상이야" 같은 정보를 주면 수준에 맞는 요약이 나와요.

**4. 긴 문서는 Claude가 유리해요.** ChatGPT보다 Claude가 긴 문서를 더 잘 처리합니다. 특히 A4 10페이지 이상이면 [Claude](/posts/claude-vs-chatgpt)를 추천해요.

프롬프트를 더 잘 쓰고 싶다면 [ChatGPT 프롬프트 완벽 가이드](/posts/chatgpt-prompt-guide)를 확인하세요.
